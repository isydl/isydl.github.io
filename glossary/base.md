## 自然现象 / 变化类
| 中文      | 低保动词                          | 说明                                     |
| ------- | ----------------------------- | -------------------------------------- |
| 出现 / 发生 | happen / start / appear       | start = 开始，appear = 看得见/出现             |
| 消失      | go / stop                     | stop = 停止存在，go = 离开/消失                 |
| 增加      | go up / get more              | 最低层表示“数量上升”                            |
| 减少      | go down / get less            | “数量下降”                                 |
| 变大      | get bigger / grow             | grow = 生长/变大                           |
| 变小      | get smaller / shrink / reduce | shrink / reduce = 变小，最低层可用 get smaller |
| 冬天带来雪   | come with                     | 自然现象带来，伴随出现                            |
| 春天带来温暖  | come with                     | 同上                                     |
| 风吹来雨    | come with                     | 风是自然主体，雨随之出现                           |
| 天变黑     | get dark                      | 状态变化                                   |

## 总结规律（可直接套用）

动作主体不同 → 动词不同

人 / 动物主动 → take / give / put / help / make

自然现象 → come / appear / start / get

抽象 / 因果 → make / get / cause + 名词/动作

方向 / 搭配 → on / in / to / away / up / down / back

修饰 → 简单形容词 + 数量词

厚厚 = thick / many / lot

快 = fast

大 = big / lot
